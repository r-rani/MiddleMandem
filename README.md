Midz helps friends coordinate meetups effortlessly by finding a fair midpoint, suggesting suitable locations, and respecting everyone‚Äôs dietary restrictions.

## Features

### Smart Midpoint Calculation
- Automatically calculates the geographic midpoint between all participants
- Ensures everyone travels roughly the same distance

### Intelligent Location & Restaurant Filtering
- Finds restaurants, cafes, and points of interest near the midpoint
- Respects dietary restrictions: vegan, gluten-free, halal, kosher, and combinations

### Location Boards
- Save favorite spots in customizable boards (Restaurants, Coffee Shops, Parks, etc.)
- Personalize boards with emojis and colors
- Locations from boards appear in meetup suggestions

### Seamless Event Creation
- Create meetup events with one tap
- Auto-generates group chat with participants
- Event details include location, date/time, budget, and dietary considerations
